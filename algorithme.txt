Partie 1 : afficher les épisodes et un épisode avec ses commentaires

Créer la base de données novel
Créer les tables :
	episodes avec les champs : id, chapter, title, content, creationDate, updateDate, userId(foreign key)
	comments avec les champs : id, author, episodeId, comment, dateComment, rudeComment

Les épisodes
	Se connecter à la base de données episodes
	TANT QUE il y a un épisode
	Aller chercher(fetch) l'épisode
	Afficher les épisodes
	Le titre de chaque épisode est un lien qui affichera l'épisode et ses commentaires

Un épisode
	Se connecter à la base de données episodes
	ALLER CHERCHER (fetch) l'épisode id = N
	Afficher l'épisode

Les commentaires de l'épisode
	Se connecter à la base de données comments
	TANT QUE il y a un commentaire de l'épisode id N 
	ALLER CHERCER le commentaire episodeId N
	Afficher les commentaires de l'épisode id N
